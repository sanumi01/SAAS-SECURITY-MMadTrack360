# MMadTrack360 Security Platform

## How to Run

1. Open PowerShell in this folder
2. Run: .\start.ps1
3. Open browser to: http://localhost:4173

## Files

- index.html - Main admin dashboard
- enterprise.html - Enterprise version
- styles.css - All styles
- app.js - JavaScript functionality

## Deployment

To deploy to S3:
1. Build is not needed (pure HTML)
2. Upload all files directly to S3 bucket
3. Enable static website hosting
